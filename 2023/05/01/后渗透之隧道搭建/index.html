配置规则

  1. iptables -t nat -A PREROUTING --dst 192.168.100.100 -p tcp --dport 3389 -j DNAT --to-destination 10.1.1.11:3389

    这个命令添加了一个PREROUTING链规则，它指定了当目标IP地址为192.168.100.100，并且传输协议为TCP，目标端口为3389时，将数据包的目标地址修改为10.1.1.11，并且目标端口保持为3389。

  2. iptables -t nat -A POSTROUTING --dst 10.1.1.11 -p tcp --dport 3389 -j SNAT --to-source 192.168.100.100

    这个命令添加了一个POSTROUTING链规则，它指定了当目标IP地址为10.1.1.11，并且传输协议为TCP，目标端口为3389时，将数据包的源地址修改为192.168.100.100。
