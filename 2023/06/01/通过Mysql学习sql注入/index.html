堆叠注入

原理

后端使用的查询函数为mysqli_multi_query() ，支持多条语句查询

而不是mysqli_query() ，仅支持一条语句查询

局限性

  • 并不是每一个环境下都可以执行，可能受到 API 或者数据库引擎的影响

  • 无回显：在 Web 中代码通常只返回一个查询结果，因此，堆叠注入第二个语句产生错误或者结果只能被忽略

    解决方法：可以通过先将内容插入到数据库中，然后再通过查询查出来
